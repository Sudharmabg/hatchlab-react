# Hatchlab Design System Rules

## Color Palette

### Primary Colors
- **Primary Yellow**: `#FFFF00` - Used for highlights, buttons, and accent elements
- **Header Background**: `rgba(23, 22, 31, 0.95)` - Used for header and main sections
- **Footer Background**: `#0b0b0b` - Used for footer and card backgrounds
- **Background Dark**: `#0f0f16` - Used for main page background
- **Text White**: `#ffffff` - Primary text color
- **Text Gray**: `#ddd` - Secondary text color

### Background Usage
- **Homepage sections**: All sections except gallery should use header background color `var(--header-bg)`
- **Cards**: Use footer background color `#0b0b0b` for better contrast
- **Gallery section**: Keep existing background (different from other sections)

## Component Guidelines

### Cards
- Background: `#0b0b0b`
- Border: `2px solid #ffffff`
- Hover border: `#FFFF00`
- Border radius: `12px`
- Padding: `30px`

### Icons
- Default color: `#FFFF00`
- Background circles: Use distinct colors (green, purple, orange) for different sections
- Size: `1.8rem` for main icons, `1.5rem` for smaller icons

### Buttons
- Primary: White background with black text
- Hover: Box shadow with white glow
- Secondary: Yellow border with transparent background

## Responsive Design
- Mobile: Single column layouts
- Tablet: Adjusted spacing and font sizes
- Desktop: Multi-column grids with proper spacing
